基于Wi－Fi的皮肤电阻变异性在线监测系统的设计

|Post Views:

论文基本信息

第一作者：董会娟 杨 圣 （中国科学技术大学工程科学学院，安徽 合肥 230031）

引用格式：[1]董会娟,杨圣.基于Wi-Fi的皮肤电阻变异性在线监测系统的设计[J].工业控制计算机,2019,32(12):119-121.

链接：

kns.cnki.net

基于Wi-Fi的皮肤电阻变异性在线监测系统的设计_董会娟.pdf

摘要

设计了一个皮肤电阻变异性在线监测系统。 该系统分为数据采集、数据实时绘制、数据处理和时频分析四部分。 结 合 Arduino NANO 和 ESP8266 sWi－Fi 模块完成数据采集、传输。 PC 端接收到数据后用 Python 编程实现在线绘图、实时更 新。 MATLAB 的小波分析工具箱提取皮肤电阻高频成分并进行时频域分析。 最终实现了皮肤电阻变异性的在线监测。

内容记录

团队：中国科学技术大学工程科学学院

问题：现有的监测系统因操作复杂、需长时间监测且以有线方式传输数据，限制了受试者的活动范围，无法实现在家庭中监测自主神经健康状况

方法：该系统分为数据采集、数据实时绘制、数据处理和时频分析四部分。 结 合Arduino NANO和 ESP8266 Wi－Fi 模块完成数据采集、传输。PC 端接收到数据后用 Python 编程实现在线绘图、实时更 新。MATLAB 的小波分析工具箱提取皮肤电阻高频成分并进行时频域分析。

效果：实现了皮肤电阻变异性的在线监测
