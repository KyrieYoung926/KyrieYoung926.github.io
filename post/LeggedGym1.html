Legged Gym Tutorial 1

Created2024-10-29|Updated2024-10-29|Knowledge
|Post Views:

Start learning from README.md

Legged Gym is a wide-used reinforcement learning framework developed by ETH Zurich.

You may refer to the official github page legged_gym

Installation

  1. Create a python env with python 3.8
  2. Install pytorch and cuda according to your device (find suitable package by conda)
  3. Install Isaac Gym and rsl_rl with intruction of official documents
  4. Install legged gym

Note: When deploying other projects based on legged gym, be aware of the version of these frameworks, which might lead to error.

Code Structure

Note:

  1. Each non-zero reward scale specified in cfg will add a function with a corresponding name to the list of elements which will be summed to get the total reward.
  2. Register new envs in envs/init.py

Others

The explanations in README.md are clear.
